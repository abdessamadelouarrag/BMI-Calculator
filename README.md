# BMI Calculator

This project is a simple BMI calculator built with React and Vite. Users can enter their height in centimeters and weight in kilograms to calculate their body mass index and see the matching health category.

## Features

- Calculate BMI from height and weight inputs
- Show feedback for missing values
- Display the BMI result with a category: Underweight, Normal, Overweight, or Obese
- Clean, centered card layout with responsive spacing

## Tech Stack

- React
- Vite
- CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
